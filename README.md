# DataV-Screen

## Project setup

### Clone repo

```bash
git clone --depth=1 http://itgit.h3c.com/Frontend-Ecology/antd-vue2-admin my-project
cd my-project
```

### Install dependencies

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
